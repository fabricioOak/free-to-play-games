# free-to-play-games

This project accsses a huge list of free-to-play games using the "Free-To-Play Games Database API" avaliable on: https://www.freetogame.com/api-doc
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
